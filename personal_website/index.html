高性能前向折叠能谱拟合框架的开发与加速实现

2023.09 – Present

独立设计 · 核心开发 · 第一作者

  • 从零设计框架架构，将"反应堆能谱 → IBD 微分截面 → 沉积能量谱 → 探测器响应 → 重建能谱"的多层卷积链路重构为四级张量算子级联，系统误差统一编码为冗余参数嵌入 χ²，整条链路在 PyTorch 计算图上端到端可微分。
  • 基于 Profiling 自顶向下定位热点（信号谱计算占 98.6%），实施三项优化：分层缓存（静态预计算 + 函数级 memoization，约 3×）、带状稀疏矩阵（CSR 存储，O(N²) → O(N·w)，约 1.5×）、分块张量化构造（块宽 512，临时张量驻留 L2，约 1.5×）。
  • JUNO only 拟合单次耗时由 110 s → 15.2 s；JUNO + DYB 联合拟合单次耗时由 50 min → 5 min，叠加张量化重构总体加速超一个数量级；双精度基准下相对误差 < 10⁻⁶，5000 组 Toy-MC 收敛成功率 100%。
  • 框架支撑了 500 万次拟合 的全量 FC 覆盖率构造，成果发表于 Eur. Phys. J. C 85 (2025) 1420，被 JUNO 合作组采纳为基础分析工具，并在第 25 届 JUNO 国际合作会议作主题报告。

技术栈：Python · PyTorch · NumPy · Linux Profiling · CSR Sparse · Tiling · LRU Cache · CPU 缓存局部性

大规模 Monte Carlo 模拟与能谱分箱策略研究

2024.01 – 2024.12

核心开发

  • 搭建支持 5,000–10,000 个伪实验的自动化 Toy-MC 流水线，覆盖 56–560 bins 多种配置；以分位数（第 16/84 百分位）非参数化定义偏差指标，规避高斯近似假设。
  • 将重建能谱 bin 宽由 20 keV 调整至 100 keV 后，关键参数统计偏差由 0.8σ 降至 < 0.2σ；模型能谱在约 300 keV 段宽处模型依赖系统误差最小（< 0.3%）。
  • 分箱与分段策略被 JUNO 首篇物理论文采纳（Nature 投稿，arXiv:2511.14593）。

技术栈：Python · NumPy · SciPy · 大规模 Monte Carlo · 协方差矩阵分析 · 参数化降维

Feldman–Cousins 统计推断框架的工程化与集群部署

2024.06 – 2025.06

核心开发 · 统计推断方案负责人

  • 将 FC 构造分解为 50 网格点 × 5,000 伪实验 × 20 多起始点 = 500 万次独立拟合，在 1,000+ 核 HPC 集群上并行编排；针对似然面多极小值采用多起始点全局优化。
  • 由经验 Δχ² 分布提取临界值 Δχ² ≈ 1.57（显著高于渐近近似 1.0），定量证实 Wilks 定理在低统计量下失效；约 8% 伪实验出现不连续置信区间。
  • 相关统计策略建议被 JUNO 首篇物理论文采纳（Nature 投稿，arXiv:2511.14593）。

技术栈：Python · PyTorch · HPC 集群调度 · 分布式并行 · 似然比检验

OMILREC 事例重建算法的加速实现

2024.06 – 2024.12

核心开发 · 性能优化负责人

  • 基于 Linux perf + Hotspot 在生产负载下进行性能剖析，定位关键热点函数并实施底层优化。
  • 引入稀疏化计算，按事例信息筛选有效 PMT、跳过无贡献 PMT；通过多核并行与几何计算向量化进一步压缩单事例耗时。
  • 将与拟合参数无关的几何量移出 Minuit 似然循环，预计算后复用，消除迭代中的重复开销。
  • 最终速度提升约 3 倍，输出与原算法逐事例一致（相对误差 ≲ 10⁻⁹），并在第 26 届 JUNO 国际合作会议作主题报告。

技术栈：C++ · Linux perf · Hotspot · Minuit · CMake
