## 评阅人意见 1

> 所提出的高性能前向折叠框架在性能加速方面优势明显，但对其在不同输入条件（如不同反应堆类型、不同统计量水平）下的数值稳定性与收敛性表现，尚缺乏系统的对比验证。

### A. 可补做的实验 / 数值测试

**A1 反应堆输入条件扫描**
- **裂变份额组合**：对以下几类反应堆输入分别运行 N=2000 次伪实验拟合——
  - JUNO 标称组合（台山 + 阳江 + 大亚湾，235U/238U/239Pu/241Pu 份额为 0.58/0.07/0.30/0.05）；
  - 单 PWR 典型堆（份额 0.564/0.076/0.304/0.056）；
  - 高钚份额 MOX 情形（239Pu 份额提升至 0.45 的假设性输入）；
  - 研究堆接近纯 235U 的输入（份额 > 0.9，模拟 STEREO / PROSPECT 情形）；
  - 以覆盖典型功率反应堆—研究堆—MOX 三端点。
- **基线 / 堆点组合**：分别测试"仅台山 / 仅阳江 / 仅大亚湾 / 全组合"共 4 档，验证框架对多点源叠加的稳定性。
- **本底配置**：关本底 / 仅地中微子 / 全本底 三档，测试在低信噪比下拟合是否仍能保持相对误差 < 1e-6 的数值精度。

**A2 统计量扫描**
- 在固定物理真值（$\sin^2\theta_{12}$、$\Delta m^2_{21}$、$\sin^2\theta_{13}$、$\Delta m^2_{31}$）下，生成统计量档次：**10 天 / 50 天 / 100 天 / 200 天 / 1 年 / 3 年 / Asimov**，每档 5000 个 toy。
- 对每档评估两类指标：
  - **收敛性**：MINUIT `fit-status` 成功率、`EDM` 中位数、`Hesse` 状态、迭代次数分布、多起点一致收敛率；
  - **数值稳定性**：相对未加速基线实现的 best-fit 值相对偏差、对数似然值差 $\Delta(-2\ln L)$、响应矩阵 $L_\infty$ 范数误差、稀疏 vs. 稠密矩阵乘法残差。
- 复用 `Chap_Binning.tex` §1.2 已搭建的 30 / 50 / 100 天 toy 基础设施，扩展至上述统计量档次。

**A3 数值精度分档基准**
- 在 FP32 / FP64 / FP64 + Kahan 补偿求和三档分别重复 A1 与 A2，量化累积舍入误差；重点验证高 239Pu 份额或极低统计量情形下 $10^{-6}$ 相对误差的守住情况。

**A4 硬件与并行可重复性**
- 同一随机种子在三类硬件上复跑一致性对比：
  - Intel Xeon Gold 6338（基线 CPU）；
  - AMD EPYC（异构 x86_64 验证）；
  - NVIDIA A100 GPU（验证张量链移植）。
- 以 bit-wise / ulp 级差异报告并行可重复性。

## 评阅人意见 2

> 该数据分析方法是否适用于其他低统计的数据分析，尚未讨论。

### A. 可补做的分析论证

**A1 核心抽象接口的梳理**
从现有框架中抽出实验无关的三类核心算子，以伪代码/函数签名形式公开：
1. **`ReactorSpectrum(phi_i, ff_i, baseline)`** —— 张量化源谱生成器，输入为各同位素基准谱与裂变份额；
2. **`BandedResponse(R, bandwidth, block_size)`** —— 带状稀疏响应矩阵构造器，处理"能量 A → 能量 B"的线性映射；
3. **`CachedLikelihood(statistic ∈ {CNP, Poisson, Gauss}, cache_policy)`** —— 缓存化似然/卡方入口，支持 LRU / memoization；
4. **`FCCoverage(toy_generator, fit_engine, n_toys, parallel_backend)`** —— Feldman–Cousins 并行化置信带构造器。
上述四类接口即为跨实验迁移的最小闭包。

**A2 典型低统计量场景的可迁移性对照**
- **反应堆短基线实验**：Daya Bay EH1–EH3、RENO near/far、Double Chooz near/far、STEREO、PROSPECT、DANSS
  - 适配项：响应矩阵维度、本底谱形、近/远点组合；
  - 预期复用率：> 80%，仅需替换输入数据与响应维度。
- **反应堆近点高分辨实验**：JUNO-TAO
  - 适配项：带宽 $w$ 显著收窄；非线性响应由分块 CDF 构造改为分段解析；
  - 预期复用率：70%。
- **无中微子双贝塔衰变**：KamLAND-Zen、nEXO
  - 适配项：剔除振荡相位模块，保留前向折叠 + FC 覆盖性模块；
  - 特别价值：0νββ 典型低统计、需严格频率学覆盖率；
  - 预期复用率：60%。
- **太阳 / 大气分析**：Super-K、SNO+ 事件分类后的低统计 bin
  - 适配项：事件分类权重、太阳天顶角依赖；
  - 预期复用率：50%，主要复用 CNP 似然与 FC 构造。

**A3 最小可复现示例（可选）**
- 以 Daya Bay 公开反应堆谱为输入，完成"EH1 单近点 → EH3 远点"的端到端拟合复现：
  - 报告：迁移所需代码行数、运行时间、$\sin^2 2\theta_{13}$ 复现值与 Daya Bay 发表值的一致性；
  - 若答辩前可完成，作为可迁移性的定量证据；否则以"迁移清单 + 工作量估计"替代。
